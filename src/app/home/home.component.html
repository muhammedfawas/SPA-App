Steps of Creating SPA App

  • 1.Creat Home,Login,Register,PageNotFound Component (ng g c new componentname)
  • 2.Import RouterModule in app.module.component.
  • 3.@ngModule->Imports->Import RouterModule.forRoot([]).
  • 4.Inside forRoot( ) Set path and component, as a object.
  • 5.Use "routrLink" set path.
  • 6.At last add router-outlet tag in app.component.html.
